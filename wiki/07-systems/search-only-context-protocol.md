# Search-Only Context Protocol

## Use When

Use this when an agent needs awareness from the vault but should not change vault files.

## Rules

- Read/search only.
- Start with [[../../index]] or the page Leo points to.
- You do not need the full vault operating model unless the task requires deeper orientation.
- Prefer reviewed/current pages.
- Treat `UNREVIEWED` material as provisional.
- Do not edit vault files.
- If exact wording matters, verify against the source page or source file.
- If something should be saved, produce a Vault Handoff instead of editing.

## Minimal Flow

1. State the mode and reason.
2. Search/read only the pages needed for the task.
3. Do the external work.
4. Run a memory check only if something durable may be worth preserving.
